## 3. Agentic AI Setup Guide (The "Gratis" Workflow)

To automate the platform for free (or low cost), we use **n8n** (self-hosted) and **Claude 3.5**.

### Step 1: Initialize n8n
- **Host:** Install n8n on a small VPS or use the Free Desktop version/Cloud.
- **Workflow:** Set a "Schedule Trigger" to fire every 15 minutes.

### Step 2: Source Collection (The Scrapers)
- Connect n8n to RSS feeds of: BBC, CNN, La Vanguardia, El Periódico.
- Use the **Google Trends (ES)** node to pull the hottest searches in Spain.

### Step 3: The "Relevance Agent" (Claude 3.5)
- **Prompt:** *"You are the GoBarcelona Editorial Lead. Rank these 50 headlines by relevance for our 50 resident personas (Student, Expat, etc.). Select only the top 3 biggest stories."*
- **Action:** This agent filters out junk news.

### Step 4: The "Editorial Agent" (Claude 3.5)
- **Prompt:** *"Rewrite this story in our 'Young Professional' voice. Create a punchy headline and 3 short paragraphs. Standardize the category."*
- **Output:** Valid JSON pushed directly to Supabase.
