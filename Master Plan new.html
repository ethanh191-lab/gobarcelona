02 The News Engine

A flagship 24/7 newsroom with zero human intervention. Inspired by NOS.nl and NOS Stories.

Sources & Scrapers

Our n8n engine pulls from 50+ localized and global endpoints every 15 minutes:

  • Global: BBC, CNN World, Reuters (filtered for Spain).
  • National (ES): El País, El Mundo, La Vanguardia, El Confidencial.
  • Local (BCN): El Periódico, BCN Secreta, Metropoli Abierta, TimeOut BCN.
  • Catalan: Ara.cat, CCMA/324 (Scraped for language/culture shifts).
  • Trends: Google Trends (Geo: ES) - Primary relevance driver.

Agentic AI Workflow ("The Editorial Meeting")

Stage 1: Relevance Scoring (The Gatekeeper)
Claude 3.5 Sonnet analyzes headlines. If a topic matches one of our 50 personas with a score > 7/10, it proceeds.

Stage 2: Editorial Rewrite (The Voice)
AI transforms dry news into "NOS Stories" style: Direct, visual, and punchy. No raw Spanish text is allowed on the EN site.

Stage 3: Visual Sourcing
Automated extraction of high-res images from sources or Unsplash API targeting "Elite Aesthetic".

04 Beer Map Architecture

The "Absolute Highlight". UX quality target: Airbnb / Booking.com.

Map Logic

  • Dynamic Proximity: 2km radius focus for active location. Always fallback to City Center.
  • Price Normalization: Everything calculated to 0.5L equivalent.
  • Color-Coding:
    🟢 Cheap (€1.8 - 3.8)
    🟡 Moderate (€3.85 - 5.85)
    🔴 Expensive (€5.9 - 10+)

Conversion & Resend Funnel

Every guide page offers a "Premium PDF Version" (Magazine Layout). To download, users provide their email. This triggers:

  1. Resend API: Delivers the PDF via transactional mail.
  2. Supabase: Adds user to the "Newsletter" segment.
  3. Affiliate: Emails follow up with the relevant referral link (e.g., SIM cards for Step 5).

07 Agent Setup (The "Gratis" Workflow)

Operating a media empire with zero human cost. Toolset: n8n + Claude 3.5.

AGENT 1: The Scraper (n8n)

Triggers every 15m. Pushes RSS/XML/HTML content to HTTP Request node. Cost: $0 (Self-hosted).

AGENT 2: The Filter (Claude 3.5)

PROMPT: "Evaluate these 50 news items. Filter for Barcelona residents. Return top 5 JSON."

AGENT 3: The Writer (Claude 3.5)

PROMPT: "Rewrite for GoBarcelona. Match Tone: Professional, Bold, Youthful. Use Syne Font rules."

AGENT 4: The Illustrator

Selects best image URL from source or generates descriptive alt-text for accessibility.

FC Barcelona Special Pipeline

When "FC Barcelona" appears in Google Trends (ES):

  • Trigger Match-Day Hero Banner.
  • Scrape Press Conferences (YouTube/Twitter).
  • AI Guide: "Where to watch Barça tonight: Best affordability vs vibe."
